Privacy Policy for MyAnalyticsTool

1. Data Collection

We collect:

  • TikTok OAuth tokens (temporary)
  • Public video metadata
  • Aggregate comment statistics

2. Data Usage

Data is used solely for:

  • Generating engagement analytics
  • Displaying historical trends
  • API-required operations

3. Data Storage

We store:

  • OAuth tokens (24 hours max)
  • No personal user data
  • No raw comment content

4. User Rights

You can:

  • Revoke API access anytime
  • Request data deletion
  • Opt-out of data collection
